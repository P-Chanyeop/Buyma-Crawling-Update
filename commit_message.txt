📊 v3.4.12: 주력상품 엑셀 업로드 & 가격분석 진행률 완전 개선

✨ 주요 신규 기능:
- 주력상품 엑셀 업로드: 상품명, 상품ID, 현재가격만으로 일괄 등록 가능
- 가격분석 진행률 정상화: 페이지별(100개 단위) 진행률 표시로 정확한 진행 상황 확인
- 10개마다 실시간 테이블 업데이트: 분석 결과를 10개마다 즉시 테이블에 반영
- 자동 스크롤: 마지막 분석 완료된 상품으로 테이블 자동 스크롤

🔧 기술적 개선:
- 엑셀/CSV 파일 지원 (.xlsx, .xls, .csv)
- 페이지별 진행률 계산으로 정확한 진행 상황 표시
- update_price_table_with_current_data() 함수로 실시간 테이블 반영
- scrollToItem()으로 마지막 업데이트 행 자동 스크롤
- 기본값 자동 설정으로 엑셀 업로드 후 바로 분석 가능

📊 사용자 경험 향상:
- 엑셀에서 수십~수백 개 주력상품 한 번에 등록 가능
- "페이지 1 분석 중: 1/100" 형태로 직관적인 진행률 표시
- 10개마다 분석 결과 즉시 확인 가능
- 테이블이 자동으로 마지막 분석 완료 상품으로 스크롤
- 엑셀 업로드 → 가격확인 버튼 클릭으로 간편한 워크플로우

🎯 업무 효율성:
- 대량 주력상품 관리 시간 대폭 단축
- 실시간 분석 진행 상황 모니터링 가능
- 페이지별 처리로 메모리 효율성 및 안정성 향상
