⚖️ v3.4.14: 가격차이 로직 통일 & 최저가 상품 수정 방지 & 엑셀 출력 개선

🔧 핵심 로직 개선:
- 가격차이 계산 통일: 모든 가격분석에서 "현재가 - 최저가" 기준으로 완전 통일
- 최저가 상품 수정 방지: 내 상품이 최저가일 때 가격 수정하지 않고 "현재가 적정 (최저가)" 표시
- price_difference = current_price_int - lowest_price로 일관된 계산

📊 사용자 경험 대폭 개선:
- 주력상품 엑셀 간소화: 상품명, 상품ID만 입력하면 되도록 현재가격 컬럼 제거
- 엑셀 출력 개선: 제안가 컬럼 제거, 가격차이를 현재가격-최저가 차이로 명확히 표시
- 상품명 정제: 엑셀 출력 시 브랜드와 상품명만 표시 (商品ID 이후 제거)

💡 직관적 가격차이 표시:
- 양수: 현재가가 최저가보다 높음 (비쌈) → 가격 인하 검토 필요
- 음수: 현재가가 최저가보다 낮음 (저렴) → 경쟁력 우위
- 0: 현재가와 최저가 동일 → 적정 가격

🛡️ 스마트 가격 관리:
- price_difference <= 0일 때 자동으로 수정 제외
- 최저가 경쟁력 유지를 위한 안전장치 구현
- 불필요한 가격 인하 방지로 수익성 보호

📈 엑셀 데이터 품질 향상:
- 헤더: 페이지, 상품명, 현재가격, 최저가, 가격차이, 상태, 처리시간
- 상품명에서 商品ID 이후 불필요한 정보 자동 제거
- 현재가격과 최저가의 직접적인 차이 계산으로 명확한 분석

🎯 업무 효율성:
- 주력상품 등록 시 현재가격 입력 불필요 (분석 시 자동 추출)
- 최저가 상품 자동 식별로 경쟁력 우위 상품 보호
- 일관된 가격차이 기준으로 혼란 없는 의사결정 지원
